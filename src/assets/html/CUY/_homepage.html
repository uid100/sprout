CS 119/119L

This class is an introduction to programming and program design using the Python programming language.

 

Each module contains lecture notes, assignments and other resources. There are coding demonstrations and exercises every week to practice the content. Each module will follow approximately one chapter from the required textbook, as well as additional content.
