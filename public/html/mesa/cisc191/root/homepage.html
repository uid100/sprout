Description (from course catalog:)

Description: This course is an intermediate level study of the Java programming language. Topics include single and multidimensional arrays; objects and classes; object-oriented programming; inheritance and polymorphism; exception handling and text input/output (I/O); abstract classes and interfaces; graphical user interfaces (GUIs); recursion; concurrency; and generic collections and data structures, such as linked lists, queues, and stacks.

This course is intended for students majoring in computer and information sciences or anyone interested in learning more about the Java programming language.

Prerequisite: CISC 190 or equivalent

Read More!

Do you want to learn more about Java? Then this course is for you! In this course, we will review the fundamentals of object-oriented programming using Java, then focus on Intermediate/Advanced concepts including inheritance and polymorphism, exception handling, text file I/O, abstract classes and interfaces, GUIs and event driven programming, recursion, multi-threading, and generic collections and data structures.
