conditional control structures

Boolean expressions and conditional control structures are fundamental concepts in programming that help you control the flow of your code based on certain conditions. In Python, these are often used with if, elif (else if), and else statements.

Boolean Expressions:

A boolean expression is an expression that evaluates to either True or False. It usually involves comparisons or logical operations.

Examples:

        x = 5
        y = 10

        # Comparison
        result1 = x == y  # False

        # Logical operation
        result2 = x < y and y > 0  # True

Conditional Control Structures:

Conditional control structures allow you to execute different blocks of code based on whether a given condition is True or False.

if statement: 
        x = 5

        if x > 0:
            print("x is positive")
        elif x == 0:
            print("x is zero")
        else:
            print("x is negative")
