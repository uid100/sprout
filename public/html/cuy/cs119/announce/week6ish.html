So the good news... tonight we did discuss (at length!):

  • how to define a method
  • how to call a method that has been defined
  • how to pass a value to a parameter for use inside the method
  • how to return a value (result) from a method

We also explained the meaning of variable scope and that variables only exist within the block of code where it was declared. And in Python, a local block of code is identified by the indentation.

 

You also saw that you can call methods from another library, or module.  There are methods in the standard Python library that you already know, like int() and str() and print() and input(), and there are methods that you have called after importing other libraries, like you did when you imported the random module and then made a call to random.next().

Notice that methods always have parentheses, and sometimes values can be included in the parentheses to pass the value to a method's parameter.
