Chapter 4
Looping

    Content:
  • Common loop structures (while, for)
  • Nested loops

 

With chapter 4, we move into the third (and last) fundamental coding structure: Loops or repetition structures. In this chapter, we'll see how we can repeat an instruction (or block of instructions) as long as some condition holds true. Just like selection structures, loop conditions must evaluate to a true/false result.

 

    At the end of this module, you will be able to:
  • Demonstrate the use of loop structures
  • Describe use-cases when for or while loops might be preferred
  • Demonstrate nesting loop blocks
