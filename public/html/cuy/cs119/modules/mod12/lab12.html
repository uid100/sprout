Exercise 1

Sorting

Design and implement a Python application that will accept an array of integers, sort it, and display the list in descending order.

Write pseudocode or you may do a flowchart to describe the process.

Hint: You may use the sort code logic that is presented in the chapter to sort string values and modify it as needed to sort integers.

IMPORTANT:

There are Python libraries which with a “sort()” method that will accomplish this but, do not use these.

The objective is to write your own function to sort and reverse the array logically, not to use the shortcut method. It is strongly recommended that you use the logic presented in the chapter.

Put your Python code to sort the array in a method called sort_integers(). Call your file Ex1Sort.py.

For example, if you are given the array:
nums1 = [12, 67, 13, 5, 45, 19, 13, 15, 23, 3]

Your program output should look something like this:
67 45 23 19 15 13 13 12 5 3

Exercise 2

Calculating Median and Mean

Design and implement a Python application that will accept an array of integers, display the array, calculate and display the mean and median values of the array.

The mean is essentially the average value of the array. Put your Python code in a user defined method called calc_mean().

Calculating the average of a list of numbers is not complex logic. As with exercise 1, write the code for these exercises, do not use built-in Python functions.

The median is the middle value of a sorted array. If an array contains an even number of elements, the median value is the mean (average) of the middle two values. Hint, the values need to be sorted first. Use your sort algorithm from exercise 1, do not use .sort()

Put your Python code in a user defined method called calc_median()

Write pseudocode or you may do a flowchart to describe the process.

Be sure to test your program with even and odd number of array elements.

For example, you may include the following 2 arrays in your test application.

  • nums1 = [12, 67, 13, 5, 45, 19, 7, 15, 23, 3]
  • nums2 = [12, 67, 13, 5, 45, 19, 7, 15, 23, 3, 32]
