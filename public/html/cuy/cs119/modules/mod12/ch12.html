Chapter Notes

Manipulating Larger Quantities of Data

This chapter provides some “advanced” coverage of arrays. In this chapter, we’ll look at some more advanced array concepts such as arrays of objects, sorting arrays, using methods to process arrays, multi-dimensional arrays and we’ll take a look at Java arrays in a little more depth.

Chapter Highlights

    Content:
  • The need for sorting data
  • The bubble sort algorithm
  • Sorting data in parallel arrays
  • Sorting objects
  • The insertion sort algorithm
  • Multidimensional arrays
  • Indexed files and linked lists
