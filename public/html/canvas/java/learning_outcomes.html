learning outcomes
# LOs
1. Employ design principles of object-oriented programming.
2. Construct programs utilizing single and multidimensional arrays.
3. Construct programs using objects and classes in object-oriented programming.
4. Construct programs utilizing inheritance and polymorphism, including abstract classes and interfaces.
5. Construct programs utilizing generic collections and data structures.
6. Construct programs utilizing Graphical User Interfaces (GUIs) utilizing event-driving programming.
7. Construct programs utilizing exception handling.
8. Construct programs utilizing text file I/O.
9. Construct programs utilizing recursion.
10. Construct programs utilizing multiple threads.
